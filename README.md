# School Backend

![School Logo](https://your-logo-url.com/logo.png)

This is the backend for a school management system.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Mahad-Ghauri/School-Backend.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file and add the required environment variables.

## Available Scripts

- `npm start`: Starts the server.
- `npm run dev`: Starts the server in development mode with nodemon.
- `npm test`: Runs tests.
- `npm run reconcile:fees:dry`: Runs the fee reconciliation script in dry-run mode.
- `npm run reconcile:fees:apply`: Applies the fee reconciliation.

The API documentation is available at `/api/docs`.
